# low-cost-oscillator-fitting
## About the Project
### This project is part of an ongoing physics research project on coupled mechanical oscillators. The goal is to computationally analyze three oscillation modes — in-phase, out-of-phase, and beats — by fitting experimental time-series data to theoretical models, and use of fft for precise computation of the normal mode frequencies. by using the data generated by a set up using expEYES, A LOW COST COUPLED OSCILLATOR

#### > Note: Real experimental data is omitted as the associated research paper is currently under review. A dummy data generator is included to reproduce the full analysis pipeline.

## some of important functions 

### np.fft.fft / rfft
### Finds frequencies in signal by Converting time-domain signal (amplitude vs time) into the frequency domain. It says about which frequencies are present and how strong each one is. stronger the frequency, longer the peak.

### df_freq.loc .loc is a label-based indexer for pandas DataFrames. Instead of using row numbers, we assign values using meaningful row names. it's being used like a results table  each row is a mode (in-phase, out-phase) and the column stores its measured frequency.

### pd.contact joins two data frames side by side (axis=1) or stacked (axis=0). Here it merges the FFT frequency table and curve-fit frequency table into one combined results table.
